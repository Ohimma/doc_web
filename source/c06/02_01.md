## 2.01 Vue 安装使用

##### 1. 本地文件 方式

在官网下载 vue.min.js 到本地，然后 scripts 直接引入        
<https://vuejs.org/js/vue.min.js>   


##### 2. cdn 方式

以下推荐国外比较稳定的两个 CDN，国内还没发现哪一家比较好，目前还是建议下载到本地             

Staticfile(国内): <https://cdn.staticfile.org/vue/2.2.2/vue.min.js>             
unpkg: <https://unpkg.com/vue/dist/vue.js>, 会保持和 npm 发布的最新的版本一致           
cdnjs: <https://cdnjs.cloudflare.com/ajax/libs/vue/2.1.8/vue.min.js>            
综合cdn平台：<https://www.bootcdn.cn/vue/>   


##### 3. npm 方式

```
$ npm -v     // 版本要大于2.3.0

$ cnpm install npm -g   // 安装或者升级到淘宝的npm镜像，方便国内开发者
$ cnpm install vue -g

$ vue init webpack my-project   // 创建一个基于 webpack 模板的新项目
$ cd my-project
$ cnpm install
$ cnpm run dev


```
