2.01 js 基本语法


##### 1. 用法结构
```
<!DOCTYPE html>
<html>
<style>
   .......
</style>
<body>
    .........
</body>
<script>
    document.write("<h1>这是一个标题</h1>");
    document.write("<p>这是一个段落</p>");
</script>
</html>
```

##### 2. 调试显示

window.alert()    弹出警告框。
document.write()  方法将内容写到 HTML 文档中。
innerHTML         写入到 HTML 元素。
console.log()     写入到浏览器的控制台。

```
<body>
    <h1>我的第一个页面</h1>
    <p>我的第一个段落。</p>
    <p id="demo">我的第一个段落。</p>
</body>
<script>
    window.alert( 5 + 6);
    document.write(Date());
    document.getElementById("demo").innerHTML = "段落已修改。";
    console.log("okok");
</script>
```
