## 2.02 css 元素选择器

##### 1. 标签选择器

```
前一节选择器 定义了 style h3标签的 为 green，那么所有的 h3 标签都会是绿色

如果我们要 h3标签 有多种颜色怎么办，就要用以下的 多类名选择器
```

##### 2. class/id 选择器
```
class:
用于描述一组元素的样式，并且可以在多个元素中使用，以 . 开头表示。
使用1 .red {color: red;}    >>>   <h2 class="red"></h2>
使用2 .red {color: red;}  .size20{font-size:30px;}  >>>   <h2 class="red size20"></h2>   多类写法

id:
用于描述一组元素的样式，并且只可以在一个元素中使用，以 # 开头表示。
使用1 #red {color: red;}   >>>   <h2 id="red"></h2>

tips：
ID属性不要以数字开头，数字开头的ID在 Mozilla/Firefox 浏览器中不起作用。

```

##### 3. 交集/并集选择器
```
交集： div.dd {} 
并集： .dd, #ii, .hh {}
```

##### 4. 后代/子代 选择器
```
后代：空格分隔，指层级下的所有，子子孙孙
子代：>关联，根据>指定子代，只包含子，不会包含其他
```

##### 5. 兄弟/相邻 选择器
```
相邻：+ 分隔，找加号前的选择器选中的 紧跟其后的 一个 符合+选中的元素
兄弟：~ 分隔，找~号前的选择器选中的 其后的 所有 符合~号选中的元素
```


##### 6. 选择器权重
```
!important  
> 行内样式(权重1000) > id选择器(权重100) > class选择器(10) > 标签选择器(1) > 通配符选择器(0)
> 标签默认样式

比如：
div ul li  === 0,0,0,3
.dddd  h1  === 0,0,1,1
```

##### 7. 举例
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title></title>
    <style>
       /* class 选择器  .开头 */
       .red {
           color: red;
       }
       /* id 选择器  #开头*/
       #green {
           color: green;
       }
       /* 交集 选择器 */
       div.dd {
           color: red;
       }
       /* 并集 选择器 ,分隔 */
       .dd,
       #ii,
       .hh {
           font-size: 10px;
       }
       /* 后代 选择器：空格分割 */
       .aaa li {
           background-color: pink;
       }
       /* 子代 选择器: > 标识 */
       .bbb>p>a {
           background-color: greenyellow;
       }
       /* 相邻元素: + */
       .ccc+div+div {
           background-color: yellow;
       }
       /* 兄弟选择器：~ 紧跟其后的所有选择器 */
       .ddd~p {
           background-color: pink;
       }
       
    </style>
</head>

<body>
    <p>=================== class选择器：  第一个会变色 ======================</p>
    <h2 class="red">叶的飘落，是风的追求，还是树的不挽留？</h2>
    <h2 class="pink">叶的飘落，是风的追求，还是树的不挽留？</h2>
    <p>=================== id选择器：  第一个会变色 ======================</p>
    <h3 id="green">叶的飘落，是风的追求，还是树的不挽留？</h3>
    <h3 id="pink">叶的飘落，是风的追求，还是树的不挽留？</h3>
    <p>=================== 交集选择器： 第一行变色且变小，第二行仅变小======================</p>
    <div class="dd">交集选择器 有div.dd 且有.dd</div>
    <h3 class="hh">交集选择器  没有h3.hh 只有.hh </h3>
    <p>=================== 并集选择器： 两行都仅变小======================</p>
    <div id="ii">并集选择器  没有div#ii 只有 #ii</div>
    <p class="dd">并集选择器 没有p.dd   只有 .dd</p>

    <p>=================== 后代选择器： 第一组00 1002会变色======================</p>
    <div class="aaa">
        <p>
            <ul>
                <li>001</li>
                <li>002</li>
            </ul>
        </p>
    </div>
    <div class="iii">
        <p>
            <ul>
                <li>001</li>
                <li>002</li>
            </ul>
        </p>
    </div>

    <p>=================== 子代选择器： 百度会变色======================</p>
    <div class="bbb">
        <p>
            <a href="#">百度</a>
        </p>
        <a href="#">谷歌</a>
    </div>
    <p>=================== 相邻选择器： 003会变色 ======================</p>
    <div class="ccc">001</div>
    <!-- <p>aaa</p>    多了个p的话，就都不会变色了，因为不相邻了-->
    <div>002</div>
    <div>003</div>
    <div>004</div>

    <p>=================== 兄弟选择器：全部标签p都会变色 ======================</p>
    <div class="ddd"></div>
    <a href="#">baidu</a>
    <p>0001</p>
    <p>0002</p>
    <a href="#">jingdong</a>
    <a href="#">taobao</a>
    <p>003</p>

</body>
</html>

```
