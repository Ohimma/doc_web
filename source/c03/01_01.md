1.01 js 基础一


##### 1. 简介
JavaScript 是 web 开发人员必须学习的 3 门语言中的一门：
1. HTML 定义了网页的内容  (人)
2. CSS  描述了网页的布局  (衣服)
3. JavaScript 网页的行为 (行为)


##### 2. 特点
```
1.  是一种轻量级的编程语言 (脚本语言)
2.  是可插入 HTML 页面的编程代码
3.  插入 HTML 页面后，可由所有的现代浏览器执行
4.  很容易学习
```


##### 3. 结构语法
```
<!DOCTYPE html>
<html>
<style>
   .......
</style>
<body>
    .........
</body>
<script>
    document.write("<h1>这是一个标题</h1>");
    document.write("<p>这是一个段落</p>");
</script>
</html>
```
