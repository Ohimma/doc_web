## 2.07 js 运算符


##### 1. 算术运算符
```
举例：y = 5

+	x=y+2    x=7    y=5	
-	x=y-2    x=3    y=5	
*	x=y*2    x=10   y=5	
/	x=y/2    x=2.5  y=5	
%	x=y%2    x=1    y=5	取模(余数)
++	x=++y    x=6    y=6	
    x=y++    x=5    y=6
--	x=--y    x=4    y=4	
    x=y--    x=5    y=4
```

##### 2. 赋值运算符
```
举例： x=10 y=5

=	x=y	 	        x=5	
+=	x+=y	x=x+y	x=15	
-=	x-=y	x=x-y	x=5	
*=	x*=y	x=x*y	x=50	
/=	x/=y	x=x/y	x=2	
%=	x%=y	x=x%y	x=0
```

##### 4. 比较运算符
```
举例： x = 5

==	  x==8	    false	
      x==5	    true	
===	  x==="5"	false	 绝对等于（值和类型均相等）
      x===5	    true	
!=	  x!=8	    true	
!==	  x!=="5"	true	不绝对等于（值和类型有一个不相等，或两个都不相等）	
      x!==5	    false	
>	  x>8	    false	
<	  x<8	    true	
>=	  x>=8	    false	
<=	  x<=8	    true


一般使用：
if (age<18) x="Too young";
```


##### 5. 逻辑运算符
```
举例：x = 6  y = 3 

&&	  and	  (x < 10 && y > 1) 为 true
||	  or	  (x==5 || y==5)    为 false
!	  not	  !(x==y)           为 true

```

##### 6. 条件运算符
```
语法：variablename=(condition)?value1:value2 

举例：
如果变量 age 中的值小于 18，则向变量 voteable 赋值 "年龄太小"，否则赋值 "年龄已达到"。
voteable=(age<18)?"年龄太小":"年龄已达到";
```
