#### 2.11 css 表格(table)

##### 1. 表格

表格 同样使用的是字体、背景等属性

```
border-collapse 属性设置表格的边框是否被折叠成一个单一的边框或隔开：
```

基本格式：
```
<table>
  <tr>
    <th>Firstname</th>
    <th>Lastname</th>
  </tr>
  <tr>
    <td>Peter</td>
    <td>Griffin</td>
  </tr>
  <tr>
    <td>Lois</td>
    <td>Griffin</td>
  </tr>
</table>

```

##### 2. 示例
