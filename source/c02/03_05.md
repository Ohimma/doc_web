## 3.05 css3 2D3D(transform)


##### 1. 2D
目的： 让某个元素改变形状、大小、位置。

**transform 属性的方法**
```
1. 定义 2D 移动
translate(x,y)	    定义 2D 转换，沿着 X 和 Y 轴移动元素。
translateX(n)	    定义 2D 转换，沿着 X 轴移动元素。
translateY(n)	    定义 2D 转换，沿着 Y 轴移动元素。
 
2. 定义 缩放转换
scale(x,y)	定义 2D 缩放转换，改变元素的宽度和高度。倍数缩放
scaleX(n)	定义 2D 缩放转换，改变元素的宽度。
scaleY(n)	定义 2D 缩放转换，改变元素的高度。
 
3. 定义 2D 倾斜旋转
rotate(angle)	定义 2D 旋转，在参数中规定角度。30deg 顺时针旋转30度
skew(x-angle,y-angle)	定义 2D 倾斜转换，沿着 X 和 Y 轴。
skewX(angle)	定义 2D 倾斜转换，沿着 X 轴。
skewY(angle)	定义 2D 倾斜转换，沿着 Y 轴。
 
4. 定义所有
matrix(n,n,n,n,n,n)	定义 2D 转换，使用六个值的矩阵。 旋转，缩放，移动（平移）和倾斜功能
```

**transform-origin 属性的方法**
```
1. x-axis: left/right/center/length/%

2. y-axis: left/right/center/length/%

3. z-axis: length
```



##### 2. 3D 

**transform 属性的方法**
```
1. 定义 2D 移动
translate3d(x,y,z)	定义 3D 转化。
translateX(x)	    定义 3D 转化，仅使用用于 X 轴的值。
translateY(y)	    定义 3D 转化，仅使用用于 Y 轴的值。
translateZ(z)	    定义 3D 转化，仅使用用于 Z 轴的值。
 
2. 定义 缩放转换
scale3d(x,y,z)	定义 3D 缩放转换。
scaleX(x)	    定义 3D 缩放转换，通过给定一个 X 轴的值。
scaleY(y)	    定义 3D 缩放转换，通过给定一个 Y 轴的值。
scaleZ(z)	    定义 3D 缩放转换，通过给定一个 Z 轴的值。
 
3. 定义 2D 倾斜旋转
rotate3d(x,y,z,angle)	定义 3D 旋转。
rotateX(angle)	        定义沿 X 轴的 3D 旋转。
rotateY(angle)	        定义沿 Y 轴的 3D 旋转。
rotateZ(angle)	        定义沿 Z 轴的 3D 旋转。
 
4. 定义所有
matrix3d(n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n)	定义 3D 转换，使用 16 个值的 4x4 矩阵。

5. 定义透视试图
perspective(n)	定义 3D 转换元素的透视视图。
```

**其他属性的方法**
```
1. 更改转换元素的位置
transform-origin: x-axis  y-axis  z-axis;  值类型 left/right/center/length/%

2. 元素在三维空间中的展现
transform-style: flat;          元素在2d平面内呈现  
transform-style: preserver-3d;  元素在3d平面内呈现 

3. 3d元素怎么查看透视图
perspective: none;  默认值，不设置透视
perspective: 10px;  元素距离视图的距离

4. 3d元素底部位置的改变
perspective-origin: x-axis y-axis;   值类型 left/right/center/length/%

5. 元素背向屏幕时是否可见
backface-visibility: visible;   背面可见
backface-visibility: hidden ;   背面不可见
```
